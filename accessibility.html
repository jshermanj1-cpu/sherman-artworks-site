Accessibility Statement

Last updated: June 2026

Our Commitment

Sherman Art Works is committed to making shermanartworks.com accessible to all users, including people with disabilities. We aim to comply with Israeli Standard 5568 (ת"י 5568) and WCAG 2.0 Level AA, as required by Regulation 35 of the Equal Rights for Persons with Disabilities (Service Accessibility Regulations), 2013.

Accessibility Features on This Site

  • All pages include a skip-to-content link for keyboard users.
  • Page language and text direction (lang / dir) are set correctly and update when switching between English and Hebrew.
  • All product images have descriptive alt text.
  • All interactive elements are keyboard-operable (navigation, product modals, gallery thumbnails, forms, cart).
  • Product modals trap keyboard focus and return focus to the trigger element on close.
  • Cart updates are announced to screen readers via a live region.
  • All form fields have associated labels.
  • Text contrast meets WCAG 2.0 AA minimum requirements (4.5:1 for normal text).
  • The site respects the prefers-reduced-motion system preference.
  • Navigation menus expose their expanded/collapsed state via aria-expanded.
  • The cookie consent banner is a labelled role="region" with accessible action buttons.
  • Page viewport is not locked - users can zoom freely.

Known Limitations

We continuously work to improve accessibility. If you encounter any barrier not listed above, please contact us using the details below.

Contact for Accessibility Issues

If you experience any difficulty using this website due to a disability, or if you find any content that is not accessible, please contact us. We will do our best to respond within 5 business days.
